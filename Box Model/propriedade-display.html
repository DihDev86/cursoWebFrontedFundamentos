Propriedade "Display"

Display Inline

O display: inline no CSS faz com que um elemento seja exibido na mesma linha que outros elementos, sem quebrar para uma nova linha. Ele só ocupa o espaço necessário para seu conteúdo, o que significa que você não pode definir larguras ou alturas específicas para ele.

Aqui estão os exemplos com as tags inline usando:

<span>: Usada para estilizar partes de texto ou agrupá-las.
<a>: Define um link.
<img>: Exibe uma imagem.
<strong>: Aplica ênfase forte (negrito) ao texto.
<em>;: Aplica ênfase (itálico) ao texto.
<input>: Cria um campo de formulário de entrada.
<label>: Associa um rótulo a um campo de formulário.
<button>: Cria um botão interativo.

Display Block

A propriedade display: block faz com que um elemento HTML se comporte como um bloco. Isso significa que o elemento:

Ocupa toda a largura disponível do seu elemento pai, independentemente do conteúdo.

Força uma quebra de linha antes e depois de si, empurrando outros elementos para cima ou para baixo. Permite definir suas dimensões (largura e altura) explicitamente usando propriedades como width e height.

Exemplos:

<div style="display: block; width: 200px; height: 100px; background-color: lightblue;">
Este é um <div> com display: block. </div> <p style="display: block; width: 150px; background-color: lightgreen;"> Este é um parágrafo (<p>) com display: block. </p>
